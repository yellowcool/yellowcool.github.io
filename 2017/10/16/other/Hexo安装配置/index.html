Hexo简介

Hexo是一个快速、简洁且高效的博客框架。Hexo使用Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

一、安装

1.安装git : sudo apt-get install git-core
2.安装NodeJs
3.安装Hexo
4.生成SSH并添加到github
5.部署项目
6.上传到github
7.绑定个人域名
8.修改及配置主题
9.添加RSS
10.添加评论

二、注意事项

  1. 当使用代码块时，要在开始的```之后加入空格跟回车，否则代码块区域会有错误。
